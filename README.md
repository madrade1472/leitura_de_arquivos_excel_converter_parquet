# Leitura de Arquivos CSV e Conversão para Parquet

Este repositório contém um conjunto de scripts e exemplos para demonstrar como realizar a leitura de arquivos Excel e convertê-los para o formato Parquet utilizando Python.

## Introdução

A leitura de dados de arquivos Excel é uma tarefa comum em muitos projetos de análise de dados e ciência de dados. No entanto, os arquivos Excel tendem a ser grandes e podem não ser a melhor escolha para armazenamento eficiente de dados. O formato Parquet, por outro lado, é altamente eficiente em termos de armazenamento e leitura, o que o torna uma opção ideal para processamento e análise de dados em larga escala.

## Funcionalidades

Este repositório oferece as seguintes funcionalidades:

1. **Leitura de Arquivos Excel:** Demonstra como ler arquivos Excel utilizando a biblioteca pandas em Python.
2. **Conversão para Parquet:** Mostra como converter os dados lidos de arquivos Excel para o formato Parquet.
3. **Exemplos Práticos:** Inclui exemplos práticos de leitura de arquivos Excel e conversão para Parquet.

## Pré-requisitos

Para executar os scripts neste repositório, você precisará ter Python instalado em seu ambiente. Além disso, certifique-se de instalar as seguintes bibliotecas Python:

- pandas
- pyarrow

Você pode instalar as dependências usando pip:
pip install pyarrow
ou
pip install fastparquet
